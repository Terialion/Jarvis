# Research Log

### [2025-05-25] Distributionally Robust Receive Combining
- **Authors**: Shixiong Wang, Wei Dai, Geoffrey Ye Li
- **Link**: https://arxiv.org/abs/2401.12345
- **DOI**: 10.1109/TSP.2025.3582082
- **Published in**: IEEE Transactions on Signal Processing (2025)
- **Summary**: Proposes a distributionally robust receive combining framework for wireless signal estimation from a statistical machine learning perspective, targeting ISAC (Integrated Sensing and Communication) systems. The framework handles multiple uncertainties (signal covariance, channel matrix, noise, impulse noise, PA non-idealities, limited pilots) and reveals that channel estimation is not necessary. Unifies existing methods (diagonal loading, eigenvalue thresholding) as special cases, and extends to nonlinear estimation in RKHS and neural network function spaces. Proves ridge/kernel ridge regression are distributionally robust against diagonal perturbation in feature covariance.
